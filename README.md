# Project smartPurse

A fintech web application to manage group savings, contributions and detect fraud using ML anomaly detection.

## Features
- Secure transactions
- Dashboard finance tracking visualizations
- MPESA Payment system
- Anomaly detection
- Easy Fund tracking
